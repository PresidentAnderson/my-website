# My Website

A modern web application built with Next.js and React.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 15
- React 19
- TypeScript

## Deploy on Vercel

This project is deployed on Vercel.
